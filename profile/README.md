# Open-Lab-Instruments
The goal of this project is to create an ecosystem of open source instrumentation, to allow every hobbyist to be able to work with instrumentation with a possible 360 degree customization.

## Announcement
This project for now is carried out only by two person, so the projects will go forward very slowly, if anyone wants to join this project contact [@mede-erik](https://github.com/mede-erik)

## Upcoming projects:
### OpenDAQ
OpenDAQ is an open-source, modular data acquisition (DAQ) system designed for maximum flexibility. Its interchangeable module architecture allows for precise measurement of voltage and temperature.
